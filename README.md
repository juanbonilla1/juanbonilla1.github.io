# Encriptador para Aula LATAM

## Descripción del Proyecto

Este proyecto es un **encriptador** diseñado para Aula LATAM. La aplicación permite encriptar y desencriptar texto mediante un algoritmo personalizado. El objetivo es ofrecer una herramienta práctica para aprender y aplicar conceptos básicos de programación y criptografía.

## Propósito del Proyecto

El encriptador fue creado como parte del proceso de aprendizaje en Aula LATAM. A través de este proyecto, se exploraron y se aplicaron conceptos fundamentales de programación, incluyendo:

- **Estructuración de código**: Creación de una aplicación web utilizando HTML, CSS y JavaScript.
- **Manipulación de texto**: Implementación de funciones para encriptar y desencriptar texto.
- **Interacción con el usuario**: Desarrollo de una interfaz de usuario intuitiva para facilitar la interacción con la aplicación.

Este proyecto no solo tiene el propósito de proporcionar una herramienta útil, sino también de demostrar el proceso de aprendizaje y comprensión de los conceptos básicos de programación. El trabajo en este encriptador ha servido como una introducción práctica a la programación, destacando la importancia de la lógica, la estructura del código, y el diseño de interfaces de usuario.

## Cómo Funciona

1. **Encriptación**: El texto ingresado en el campo de entrada es encriptado utilizando el algoritmo implementado en JavaScript.
2. **Desencriptación**: Permite la conversión del texto encriptado de vuelta a su forma original.
3. **Interfaz de Usuario**: La aplicación presenta una interfaz clara con instrucciones sobre cómo usar las funciones de encriptación y desencriptación.
